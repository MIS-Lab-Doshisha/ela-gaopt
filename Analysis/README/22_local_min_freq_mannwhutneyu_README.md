# Usage Guide: `22_local_min_freq_mannwhutneyu`

This script performs group comparison of local minima frequency distributions between two groups (e.g., ASD vs CTL) using Mann-Whitney U test. It calculates effect sizes, applies FDR correction for multiple testing, and saves results as CSV files for each individual ROI set.

- **Location:** Analysis/
- **Tested with:** Python 3.13.2

## Overview
`22_local_min_freq_mannwhutneyu.py` compares local minima frequency distributions between two diagnostic groups. The script:
1. Loads group labels from participant TSV files
2. Loads local minima frequency data from pickle file (generated by `21_local_min_freq_calc.py`)
3. For each individual (ROI set), performs Mann-Whitney U test between groups for each local minimum state
4. Calculates effect size r for each comparison
5. Applies FDR correction (Benjamini-Hochberg) for multiple testing
6. Saves detailed results as CSV files for each individual

## 1. Inputs
- **Local minima frequency data** (`freq_data_path`): Pickle file containing frequency distributions for each individual (output from `21_local_min_freq_calc.py`). Expected format: list of DataFrames with frequency data per subject.
- **Group labels**: TSV files in the data directory containing participant information. Expected format: `participants_{dataset}.tsv` with `dx_group` column (1 for group 1, 2 for group 2).

## 2. Outputs
CSV files saved to the specified output directory:
- `local_min_group_comparison_{i+1}.csv` — Statistical test results for individual i+1

### Output Structure (per CSV file):
- **Column**: Local minimum state identifier
- **Statistic**: Mann-Whitney U statistic
- **P-value**: Raw p-value from the test
- **r**: Effect size r (z / sqrt(N))
- **FDR_P-value**: FDR-corrected p-value

## 3. Configuration (script-level variables)

Located in the `main()` function:

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `pkl_path` | string | `"Data//test_data_2//binary_data"` | Directory containing participant TSV files with group labels |

## 4. Behavior / Workflow

### Main Workflow:
1. Load group labels from all participant TSV files in the data directory
2. Identify indices for group 1 (dx_group == 1) and group 2 (dx_group == 2)
3. Load local minima frequency data from pickle file
4. Call `group_comparison()` to perform statistical tests for each individual
5. Print example results for individuals 79 and 98

### Key Function: `group_comparison(data, group1_indices, group2_indices, output_dir)`

**Purpose**: Perform Mann-Whitney U tests for all individuals and save results

**Steps** (per individual):
1. Extract frequency data for group 1 and group 2 subjects
2. For each local minimum state (column):
   - Perform Mann-Whitney U test between groups
   - Calculate effect size r: `z / sqrt(N)` where z is standardized U statistic
   - Store raw p-value, statistic, and effect size
3. Apply FDR correction to all p-values for the individual
4. Save results as CSV file
5. Return list of result DataFrames

### Key Function: `load_group_labels(pkl_path)`

**Purpose**: Load diagnostic group labels from participant files

**Steps**:
1. Scan directory for pickle files
2. For each pickle file, find corresponding TSV file (participants_{name}.tsv)
3. Load `dx_group` column from TSV
4. Concatenate all group labels into a single Series
5. Return group labels indexed by subject

## 5. Usage examples

### Basic usage (compare groups for all individuals):
```bash
python Analysis/22_local_min_freq_mannwhutneyu.py
```

The script will load data, perform tests, and save CSV files for each individual.

### Change group definitions:
Edit the group assignment in `main()`:
```python
group1_indices = id_df[id_df == 0].index.tolist()  # Different group coding
group2_indices = id_df[id_df == 1].index.tolist()
```

### Point to different data:
```python
pkl_path = "Data/your_dataset/binary_data"
freq_data_path = "ELAGAopt_result/Analysis_result/local_min_freq_data/freq_your_data.pkl"
output_path = "ELAGAopt_result/Analysis_result/local_min_group_comparison_results_your_data"
```

### Change FDR correction method:
Edit the `group_comparison()` function:
```python
reject, pvals_corrected, _, _ = multipletests(results_df['P-value'], alpha=0.05, method='bonferroni')  # Bonferroni instead of FDR
```

### Modify effect size calculation:
The script uses effect size r. For other effect sizes, modify the calculation:
```python
# Add Cohen's d calculation
d = (group1_df[column].mean() - group2_df[column].mean()) / ((group1_df[column].std() + group2_df[column].std()) / 2)
```

### Statistical measures:
- **Mann-Whitney U**: Non-parametric test statistic comparing distributions
- **P-value**: Probability of observing the data under null hypothesis
- **r**: Effect size (correlation coefficient equivalent for Mann-Whitney)
- **FDR_P-value**: P-value corrected for multiple testing using Benjamini-Hochberg method

## 7. Notes & recommendations

- **Multiple testing correction**: FDR correction is applied per individual to control false discovery rate across local minimum states.
- **Effect size interpretation**: r values: 0.1 = small, 0.3 = medium, 0.5 = large effect.
- **Group balance**: Ensure groups have sufficient sample sizes for reliable statistical tests.
- **Data format**: Frequency data should be properly formatted from `21_local_min_freq_calc.py`. Missing values are filled with 0.
- **File naming**: Output files are numbered starting from 1 (individual 0 becomes file 1).
- **Performance**: Processing many individuals with many local minima states can be time-intensive.
- **Reproducibility**: Results depend on group assignments and data ordering.

## 8. Troubleshooting

### Issue: FileNotFoundError for pickle file
- **Cause:** `freq_data_path` is incorrect or `21_local_min_freq_calc.py` has not been run
- **Solution:** Verify that the frequency data pickle file exists and the path is correct

### Issue: FileNotFoundError for TSV files
- **Cause:** Participant TSV files not found or naming pattern mismatch
- **Solution:** Check that TSV files follow pattern `participants_{dataset}.tsv` and contain `dx_group` column

### Issue: "ValueError: could not convert string to float"
- **Cause:** Group labels contain non-numeric values
- **Solution:** Ensure `dx_group` column contains only 1s and 2s (or your defined group codes)

### Issue: Empty result DataFrames
- **Cause:** No overlapping local minimum states between groups or data loading issues
- **Solution:** Check that frequency data contains the expected columns and groups have data

### Issue: All p-values are 1.0
- **Cause:** Groups have identical distributions or very small sample sizes
- **Solution:** Verify group assignments and check sample sizes per group

### Issue: FDR correction fails
- **Cause:** No valid p-values to correct or statsmodels version issue
- **Solution:** Check that p-values are numeric and within [0,1] range

### Issue: Memory error during processing
- **Cause:** Large frequency datasets loaded into memory
- **Solution:** Process individuals in smaller batches or increase system memory

### Issue: Inconsistent results across runs
- **Cause:** Data ordering issues or random operations
- **Solution:** Ensure consistent data loading and check for any sorting operations

### Issue: Effect size r calculation errors
- **Cause:** Division by zero or invalid statistic values
- **Solution:** Add error checking:
  ```python
  if std_U > 0:
      z = (t_statistic - mean_U) / std_U
      r = z / N**0.5 if N > 0 else 0
  else:
      r = 0
  ```

### Issue: Output directory not created
- **Cause:** Permission issues or invalid path
- **Solution:** Create directory manually or check write permissions

### Issue: ImportError for statsmodels
- **Cause:** statsmodels package not installed
- **Solution:** Install statsmodels:
  ```bash
  pip install statsmodels
  ```
